《红楼梦》读后感

发表于 2023-10-02 分类于 读书笔记
本文字数： 2.2k 阅读时长 ≈ 2 分钟

我在很小的时候就听闻《红楼梦》大名，小学、初中、高中语文老师无不喜爱这本巨作。在小学时，学到课文中林黛玉进贾府的片段，也仅仅只是惊叹于作者描述林黛玉、王熙凤时高超的技巧，感叹作者词藻华丽、语言内涵丰富。到了初中，随着年龄的增长，慢慢梳理清楚红楼梦的故事梗概、人物关系以及其反应的社会背景。本可以细细琢磨作者的“满纸荒唐言”，但迫于学业压力，终究把阅读它的任务搁浅。如果不是此次选修课的作业要求，我不知何时才有雅兴细细阅读这本名著。

很难说我在学期内有很多时间仔细品味这本书，也很难说我明白了这本书中蕴含的哲理。如今读《红楼梦》，也有许多粗读、略读之处，在读的过程中，遇到许多诗歌仍然会选择跳过，但既然认真读了，我还是简单谈一下自己的感想，不全面，但也是我自己内心真实想法的表达。

《奥本海默》观后感

发表于 2023-10-02 分类于 观影感悟
本文字数： 1.9k 阅读时长 ≈ 2 分钟

传记类型的电影相比于其他电影来说会“小”一些，这种小指的是故事情节方面。这部电影不同于一些大片，那些片子通常是宏大叙事，它们构建的故事往往是组织与组织之间、国家与国家之间的 一种或多种冲突与矛盾，牵涉范围也会极大极广，剧情设计以及工具使用会给人一种非常震撼的感觉。

但《奥本海默》不一样，因为它是个人的传记片。一个人无论自身经历再怎么波澜起伏，震撼性肯定是远比不上那种宏大叙事的、大场景、大制作的片子。这部片子的高潮部分没有原子弹实地爆炸（而是试验）的视听盛宴、没有超级英雄开挂般的成长经历。它的情节也不是不断抛问题设悬念，之后让观众恍然大悟的层层递进式效果。其实观众看很多情节都可以大致猜到后面会发生什么。整部片子就是原原本本地讲了一个人的故事：奥本海默人生当中一个极其关键的阶段发生的故事。

《唐宋词十七讲》摘抄

“明月不谙离恨苦，斜光到晓穿朱户。”他说月亮慢慢地西斜，它的光线从朱红色的门窗照进来，我一夜无眠，看到月光从深夜直到天明。

我们有一句俗话说，观人于揭让，不若观人于游戏。我们要观察一个人，看他揭让进退，他在一个公开的场合，表现得彬彬有礼。当然，因为那是要给大家看的。可是，当他游戏的时候，他眼人赌博，一输钱就急了，什么都流露出来了。游戏之中，因为他不用端着架子，不用装腔作态了，所以，不知不觉反而把他更真的自我表现出来了。词就是有这样一种微妙的作用。

从公路小说看美国的历史与社会——《美国众神》读后感

发表于 2023-10-01 更新于 2023-10-02 分类于 读书笔记
本文字数： 3.1k 阅读时长 ≈ 3 分钟

我是一个科幻小说爱好者，喜欢阅读高质量的科幻小说，欣赏既具有科学思维又怀有文学素养的作家。在疫情期间，我阅读了《美国众神》这本书，挑选这本书阅读最初是因为它获得了“星云奖”等诸多科幻大奖。但当我真正开始阅读后，发现它和我心中的“科幻”类型小说几乎沾不上边，只能说它是一本幻想小说，更确切一些，它是一本移步换景的公路小说。

广义的科幻当然包括幻想小说这一类，经过一周的时间，我耐着性子把它看完了。作者的写法和许多外国作家一样，初读时会觉得有些地方写得比较拖沓，读起来味同嚼蜡。但我读完这本书后，感觉通过这本书我似乎实地体验了美国的风土人情。通过作者的描写，我仿佛对美国社会构成、美国的历史、美国人的价值观有了更深刻的了解。
